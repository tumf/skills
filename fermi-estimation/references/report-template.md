# Report Template

Use this default structure for a persuasive Fermi estimate.

## Template

```markdown
# [Question]

## Bottom line
- Best estimate: [value]
- Plausible range: [low] to [high]
- Scope: [geography, period, unit]

## Model
- Formula: [driver A] x [driver B] x [driver C]
- Why this model fits: [one short paragraph]

## Inputs and evidence
| Driver | Low | Base | High | Basis | Source |
| --- | ---: | ---: | ---: | --- | --- |
| ... |

## Calculation
- Show the arithmetic in one compact block

## Sanity checks
- Check 1: [result]
- Check 2: [result]

## Sensitivity and confidence
- Biggest uncertainty: [driver]
- Confidence: low / medium / high
- What would improve this estimate most: [specific missing data]
```

## Precision Rules

- Round the final answer to 1-2 significant digits unless precision is strongly justified
- Keep intermediate arithmetic more precise than the headline number
- Prefer "about 120,000" over "119,384" unless exactness matters

## Tone Rules

- Be direct about uncertainty without becoming vague
- Separate sourced facts from assumptions
- Explain why the final estimate is useful despite uncertainty
