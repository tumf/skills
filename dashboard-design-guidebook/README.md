# Dashboard Design Guidebook

デジタル庁のダッシュボードデザイン実践ガイドブックをもとに、見やすく判断しやすいダッシュボードを設計・改善するためのスキル。

## カバー範囲

- 提示型ダッシュボード向けの要件整理
- 情報選定、骨格レイアウト、グラフ選定
- 色・コントラスト・アクセシビリティ要件
- 実装前後のチェックリスト
- 軽量HTML、Grafana、SQLiteベース画面への適用

## 使いどころ

- 新しいダッシュボードを設計したい
- 既存ダッシュボードの見づらさを改善したい
- KPI、比較軸、配置、グラフ種別をレビューしたい
- 行政・業務向けの情報提示画面を整理したい
