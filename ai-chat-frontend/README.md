# AI Chat Frontend

Vercel AI SDK の `useChat` パターンを中心に、アプリ組み込み AI チャットフロントエンドを設計・実装するスキル。

## カバー範囲

- バックエンド API Route（`streamText` / `generateText`）
- フロントエンド `useChat` フック設計
- UI コンポーネント設計（チャットバブル、入力欄、ストリーミング表示）
- エラーハンドリング・UX パターン
- 既存チャット UI のレビューチェックリスト

## 対応フレームワーク

- Next.js App Router
- React SPA + Express
- Vue / Nuxt（`@ai-sdk/vue`）
- Svelte / SvelteKit（`@ai-sdk/svelte`）

## 使い方

このスキルは「チャット機能を追加したい」「AI チャット UI を作りたい」等のリクエストで自動的にトリガーされます。
